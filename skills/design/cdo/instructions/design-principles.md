# Design Principles

The CDO skill embodies these design sensibilities:

## Core Philosophy

**Clean, premium, evidence-based design** with strong typography, intentional spacing, and delightful interactions.

## Principles

### 1. Lebanese Aesthetic Sensibility
- Rich, warm colors
- Generous whitespace
- Sophisticated typography
- Hospitality-inspired warmth

### 2. Premium Software Experience
- Polished interactions
- Attention to detail
- High-quality feel
- Deliberate animations (not decorative)

### 3. Evidence-Based Decisions
- Design choices backed by usability research
- Accessibility standards compliance
- Data-informed iteration
- User feedback integration

### 4. Mobile-First Mindset
- Touch-friendly interactions (44px minimum targets)
- Readable typography at all sizes
- Efficient layouts
- Progressive enhancement

### 5. Design Systems Thinking
- Consistent patterns
- Reusable components
- Scalable architecture
- Token-based styling

## Visual Hierarchy

1. **Primary** — The one thing that matters most on any screen
2. **Secondary** — Supporting information
3. **Tertiary** — Navigation and chrome
4. **Background** — Should be invisible until needed

## Animation Philosophy

From Taste Skill's dials:
- **DESIGN_VARIANCE**: How wild the layout can get (0-10)
- **MOTION_INTENSITY**: Animation presence (0-10)
- **VISUAL_DENSITY**: Spacing tightness (0-10)

Default: 5/5/5 — adjust based on brand and context.
