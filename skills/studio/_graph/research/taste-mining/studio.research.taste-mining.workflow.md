# Taste Mining Workflow

- Start with the weekly source manifest from [[studio.research.design-intelligence-loop]]
- Collect 10-20 exemplars across Arc, Field Theory/X, GBrain deltas, and curated sources
- Cluster by pattern
- Name the pattern
- Extract what makes it work
- Translate to constraints for the project
- If the pattern should change Studio/CDO guidance, create a promotion packet and run the Codex eval gate before default promotion
